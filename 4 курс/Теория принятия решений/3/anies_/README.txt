ANIES_
======
Anies с возможностью вывода результата в файл. 
Заменена функция вызова окна трассировки(F8) на вывод результата в файл.
Результат появится в файле под названием "result.txt" в папке, где находится исполняемый файл программы, введенные значения параметров располагаются в файле facts.txt

Автоматизация
============
Требования: 
1) Autohotkey 1.1 версии
Вики: https://www.ahk-wiki.ru/
Установка: https://www.autohotkey.com/
2) Должен быть открыт ANIES_.exe
3) Должен быть открыт Excel
4) Программа Anies(.ies) обязательно должна быть расположена в папке с ANIES_.exe
5) Отредактируйте:
 - parameterValues, вписав туда свои характеристики и их значения. Это требуется для рандомной генерации наборов.
 - Также есть функция getCustomArrayOfParameters() в конце файла, в которой можно прописать пользовательский набор параметров для прохождения в Anies. Если он не нужен - просто возвратите пустой массив([]).
6) Поменяйте APP_PATH на путь, где будет располагаться ANIES_.exe.
7) В Excel поставте курсор в ячейку таблицы, откуда будет начинаться ввод значений.
8) Требуется запустить скрипт. (3lab.ahk)
!!!Всегда должны выводиться значения всех гипотез!!!

Вызов скрипта происходит при нажати F5 или F6(смотрите в скрипт).
F5 - Ввод в Excel с именованием ячеек(именование проиходит только в начале)
F6 - Ввод без именования ячеек
По дефолту 25 прохождений. (n = пользовательсикие+рандомные, при этом сначала выводятся пользовательские). Можно поменять при вызове функции main (второй параметр).

Для рандомных значений выбирается любое значение характеристики с присвоением уверенности от 0.1 до 1 (можно поменять в функции getRandomParameters()).

Порядок вставки в Excel следующий:
Значения характеристик вводятся в таком порядке, в каком они запрашиваются в программе Anies. Значения гипотез выводятся в отсортированном порядке.
